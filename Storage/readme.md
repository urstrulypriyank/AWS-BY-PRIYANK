# Storage Options In aws 
In AWS we have 3 types of storage service
- Block Storage
- Object based storage
- File based storage
  
Based on above classification of storages we have services provided by aws 

- BLOCK STORAGE
    - Amazon EBS
    - EC2 instance store
- OBJECT STORAGE
    - Amazon S3
    - Amazon GLacier
- File Based Storage
    - Amazon EFS
    - Amazon Fsx
